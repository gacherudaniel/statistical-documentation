
# 3.7 Finalise production systems

This sub-process includes the activities to put the assembled and configured processes and Services, including modified and newly created services, into production and ready for use. The activities include:
- Producing documentation about the process components, including technical documentation and user manuals;
- Finalizing process instruments and systems to be used;
- Training the users on how to operate the process;
- Moving the process components into the production environment and ensuring they work as expected in that environment (this activity may also be part of sub-process [3.5](3.5.md)).

## Input

- ## Output

- Finalized instruments and systems
- Process documentation
- Training of users
