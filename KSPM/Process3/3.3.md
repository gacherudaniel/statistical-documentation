
# 3.3 Build or build dissemination components

This sub-process describes the activities to build new components or reuse existing components needed for the dissemination of statistical products as designed in sub-process 2.1 (Design outputs). All types of dissemination components are included, from those that produce traditional paper publications including library services to those that provide web services,(linked) open data outputs, geospatial statistics, maps, or access to microdata/metadata.

## Input

- [Output from phase 2](../Process2/2.md)
- Dissemination Manuals and guidelines

## Output

- Schedule of the release of publications
- Dissemination plan
